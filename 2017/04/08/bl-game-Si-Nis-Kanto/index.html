Si-Nis-Kanto至上之空食完感想

游戏
  • BL
  • 腐女世界

AVG蔷薇向游戏里黑手党类的游戏真是不少啊, steal啊，沉默法则啊（揪心，一直没玩完)…..Si-Nis-Kanto里的黑手党之争，勾心斗角，尔虞我诈让我看的好过瘾啊，主角小受也还可以，比较别扭傲娇，但是有担当！

讲真，这个游戏的配乐啊，画风啊都挺喜欢的，但是大量重复感的cg让人感觉不太走心啊！
第一条推的线是卡尔洛线，使我大为惊艳，迅速拉高了我对这个游戏的期待值，然后满心欢喜的去推了另外两人雄午，艾西卡，就有点失望了。卡尔洛様，大好きだ！

游戏一开始的时候，我就看到了蛮区东边势力黑莲会的三位（雄午，千冬，周），以及西边的势力大头ZENCA(卡尔洛，伊格，瓦伦丁)，这两对人马一上来就互懟，还怼出点暧昧出来。比如卡尔洛调戏炸毛的雄午，我还真希望有个结局是他俩一起，毕竟卡尔洛对雄午有着特殊的执着哟，貌似艾西卡线里两方最后结成同盟，同为首领的他们也许未来发展出点什么也未必不可能哟。

至于小受真纪这边嘛，一开始是他和艾西卡在校园里的情景，总觉得和那两位大佬的生活格格不入，后来发现真纪从小和母亲相依为命，某天母亲被黑手党杀死，他最后辗转被收进了接纳孤儿的财团掌控的“慈善校园”，就觉的其中肯定有什么猫腻，他的身世绝对和某个黑手党组织有关。必然是作为黑手党的父亲为了妻子和孩子的安全，离开他们，但妻子和孩子仍然因此被卷入了斗争这类的情节。

起因是，亨利 吴，与蛮区未来息息相关的伟人，因心脏衰竭，生命受到威胁，黑莲会作为一个秉持道义的组织，其首领伊贺是亨利几十年的老友。失去亨利的领导，蛮区等于失去了未来。为了避免这件事，黑莲会打算违背一次道义，秘密地寻找和亨利匹配的人进行心脏移植为亨利续命。

亨利如果死了，东区的黑莲会地位也摇摇欲坠，为了扩大势力，西区的各位必然会阻挠黑莲会，首当其冲的就是ZENCA的卡尔洛他们，当然卡尔洛本人对阻挠雄午的好事也是非常乐意的。

之前看到卡尔洛的人设，朋友说有一股牛郎店的画风，我虽然不以为然，却不料被他说中了。

狂拽酷帅的坏坏boss－－卡尔洛

这个卡尔洛的人生历程可以说是一部心酸血泪史啊，身为娼妇的儿子，很小就被逼着去接客，但是卡尔洛逃了出来，找教堂的教父庇护自己，他深深明白，要活下去，要掌握自己的命运，需要知识，需要力量（知识就是力量），于是他不断学习，同时发现和一些商人，政客进行身体交易可以换来一些好处。

还在教父那里的时候，碰到了雄午，觉得那时的他虽然没头脑，又莽撞，但是敢说敢做，内心憧憬着他。所以对后来雄午加入黑莲会做少当家，认伊贺做爹，“做别人的狗”这件事非常不爽。他之所以老跟雄午对着干，就是想逼他脱离黑莲会。

卡尔洛后来被ZENCA的原boss看上了，当了他的情人／男妓之一，十分疼爱，甚至为了卡尔洛的安全，那位首领收他做了义子。但因为太过沉迷卡尔洛，每天只想跟他在一起，毫无斗志，再这样下去ZENCA会完蛋的，最后卡尔洛杀了boss, 自己做了首领。

是不是觉得卡尔洛很有心机，很绝情呢，是啊，就是这么一个不择手段，没有感情的人，唯独对某个人柔情，心软的时候难道不是尤其感人嘛。

卡尔洛不信任别人，包括自己的两个心腹，对其他人也是动不动就拔枪相向（拔枪好帅），而且能屈能伸。

具体他和真纪的爱情发展路线（很甜），大家可以自己去推good end，我就列出几个我比较喜欢的地方啦。

  • 瓦伦丁说卡尔洛之前明明都不介意被别人看到他和首领滚床单，但是和真纪一起的时候，就让手下别打扰
  • 卡尔洛为了宣誓自己对真纪的主权吧，邀请“觊觎”真纪的爱德和他俩一起玩了一场略微sm的3p，但是只准爱德摸前面，和真纪接吻或者后面一律不许，哎呀，你这大傲娇，谁叫真纪对爱德笑的那么欢，受罪了吧。结果此举触了真纪的底线吧，气的拿枕头砸他，然后卡尔洛乖乖被砸，让他滚出去就真的出去了。试想，真纪你可是被抓来的啊，卡尔洛想杀你分分钟的事儿啊。
  • 卡尔洛明明警告真纪，如果交战中他被黑莲会的人抓走，他一定会当场杀了真纪，绝不给黑莲会保住亨利的机会。然而结果呢，真纪鲁莽，真被抓住了，卡尔洛是怎么做的呢。他骑着自己二轮的爱车从火海中奔出来，甚至不惜用相撞这种同归于尽的方式停下了黑莲会的四个轮子。真纪还担心他翻了的小车呢，结果人家卡尔洛已经神不知鬼不觉的拿枪抵着雄午逼他放人了，帅炸啊！他为了真纪，爱车报废了，自己也一只腿踏进棺材了，最后总要给自己组织一个交代吧，竟然只罚了真纪擦一个星期还是一个月的鞋，什么？这惩罚，绝对是袒护包庇啊！
  • 还是上面那件事，好像炸了某组织（忘了名字）的仓库，那头生气了，直接老大来问罪了，还说非要ZENCA交出那个器官提供者——真纪才肯罢休。但是卡尔洛维护真纪啊，一口咬定，那个人已经被他玩完丢了。可是他手下伊格看老大被盘问的那么紧，自作主张把真纪带过去了，卡尔洛可生气了，耳语伊格一会儿找你算账。最后为了平息那头的愤怒，卡尔洛答应了那老头SM的要求，被鞭子抽啊！一声不吭，唯一吭的几声还是为了门口离开的真纪打掩护。
  • 还有很多处，卡尔洛总让自己的手下把真纪的安全放在第一位，不要管自己。真是可怜了伊格和瓦伦丁，明明是救他心切，还被骂了。
  • 卡尔洛受了重伤，需要立即手术取子弹，但是不肯麻醉，他怕自己失去意识，不能战斗，就没人保护真纪了，因为他谁都不相信。不过真纪还是把他弄昏了，再不休息，卡尔洛真的会流血过多死掉的。这边也蛮让人心疼，卡尔洛总以为让部下看他他的虚弱，软弱，大家就会离他而去，甚至背叛他，可是真纪知道，他的部下中很多都是出于对他的崇拜，远比他想的要喜欢他的多。不过后来卡尔洛也在真纪的感染下，学会相信他人，借助他人的力量了。
  • 真纪被威胁要帮别人咬，看着那人抵在昏迷的卡尔洛头顶上的枪，真纪委曲求全，正准备开始呢，卡尔洛突然醒了，然后迅雷不及掩耳之势削了那人的头盖骨（叫你调戏老大的老婆）。卡尔洛一醒来看到自己在陌生的地方，就知道自己又被背叛了，一睁眼就看到真纪没有不情愿的帮别人那个，那心里那小醋吃的哟，他说，明明都没有对我那样过。真纪可委屈了，本来被人背叛就够伤心的了，然后还被那样威胁，换作以前，他宁愿死，可是他不想卡尔洛死啊，明明是抓自己的坏人，明明是自己一直想杀死的人，可是就是不想他有事。花了多大的决心才忍受下来，人家不领情还误会他，当场气的要拔枪自尽，然后卡尔洛语气立马软了下来，这边好棒啊
  • 还有真纪为了保护卡尔洛中枪那段，卡尔洛整个人都懵了，明明还在枪战，都忘了防备，就一心想接住真纪的身体，还开了个大，大开杀戒逃了出去。甚至还脑子不清醒的想单枪匹马杀进黑莲会给真纪拿救命的血液。
  • 被爱德警告继续留在卡尔洛身边会给他带来危险的真纪，多番考量下，决定离开卡尔洛，他不打算偷偷溜走，因为卡尔洛一定会把他找出来，于是他决定进行卡尔洛和他当初的那个约定，在期限内，只要他能用弹簧刀让卡尔洛流下一滴血，就放他和艾西卡走。一开始的时候，哪怕做爱的时候，卡尔洛都是防备着的，根本下不了手。然而这次，卡尔洛已经不能像以前那么从容了，真纪格外的主动让他意乱神迷，于是就华丽丽的被刺了。这里心疼真纪，明明他最想教卡尔洛相信别人，但自己却不得不背叛卡尔洛，以后卡尔洛再也不会相信任何人了吧。卡尔洛愤怒的掐住了他的脖子，让他离开自己，卡尔洛宁愿他死在自己手里吧，可是最后还是不忍心，放了他。这之后，卡尔洛越来越残暴，手下们都看不下去了。不过还好最后真纪回来了，对啊，我们又不是当事人，怎么能擅自决定这样对对方就是好的呢。回来的真纪，卡尔洛都没有问他原因，只是格外珍惜，应该真的是被吓到了吧。他自己说，他对骑乘位都有阴影了，真是笑死我了。

还有很多细节吧，我都想再走一遍了，卡尔洛本身的人物魅力很大啊！喜欢！
至于悲剧的结局嘛，无外乎卡尔洛死了，或者丧失斗志，然后ZENCA没落。还有一个好像是真纪成了男娼，又回到卡尔洛身边，卡尔洛被迷住了…..

卡尔洛线有个台词我喜欢

既然要背叛我，就不要说喜欢我，既然说了喜欢我，那就不要背叛我啊

耿直又猛兽的少当家－－雄午

雄午我真是不好说他了，把漂亮的人都当女孩，小时候见到的真纪啊，卡尔洛啊，还有伊利亚。这边算是回忆梗还是青梅竹马梗。反正就是真纪和雄午（小时候化名莲）儿时在一起，真纪的伤就是为了保护雄午弄来的，然后雄午作为一个有担当的男子，就给了真纪一枚戒指，说要和他在一起（就是想和他结婚）。

黑莲会是个正直的帮会，虽然是黑手党，对真纪还是以礼相待。什么都好说，愿意实现他的一切愿望，但就是放真纪走这件事没的商量。周和千冬其实都知道真纪就是雄午一直在找的那个“新娘子”，但是两人深知雄午的执着，所以都瞒着不说。而真纪心里其实很讨厌黑手党，所以也不认为儿时的莲会变成黑手党，也就没怀疑雄午。

即便如此，雄午还是对真纪很有好感（很想跟他做一次），哈哈哈。讲真，这里我和真纪一样的感觉，雄午明明说中意自己，喜欢自己，可是却毫不犹豫的要自己去死。弄的真纪非常矛盾和委屈。

雄午线的剧情还是不够的，中间有个雄午发狂的情节，不知道有啥作用，倒是体现了千冬对雄午的关爱，驯化野兽，为己所用，也承担着野兽噬主的风险。被发狂的雄午强上了的真纪也抗拒了几天，但最后原谅的很快啊。

我比较喜欢的地方就是，知道真纪就是小时候和他约定的真纪之后，雄午义无反顾的背叛了黑莲会，要救真纪出来。

他其实为黑莲会卖命，为亨利找供体，也只是以为伊贺对他有恩而已，但真纪也于他有救命之恩。可以说，如果真纪没有儿时的这一层身份，雄午是绝对不会救他的。所以，我也不知道这是好还是不好，到底是出于恩情还是爱情。身份没戳破的时候，雄午明明说喜欢真纪，但是谈到儿时的真纪还是一脸幸福，完全没有出轨的意识，真纪也很不理解。

说到底，这就叫执着？？无论谁都比不上小时候那个救了他的真纪吧。

有个结局是，真纪为亨利牺牲了，然后雄午在真纪的遗物里发现了那枚戒指。这里就没剧情了，留给玩家无限的遐想，不过我猜雄午会发狂吧！

雄午线里，我就怀疑真纪是亨利的儿子了，头发颜色一样，血型一样嘛。

两面性格的小哥哥－－艾西卡

艾西卡是真相线，他从一开始就知道校园慈善都是假象，不过是财团的交易，这些孩子们长大之后都是要被贩卖出去的。而他本人因为和财团有密切的关系，一直平安无事。很早，他就开始喜欢真纪了，从那之后，便开始谋划如何让真纪逃出去吧。

艾西卡比较让我喜欢的一点是他的两面派，在别人面前和真纪面前完全就是两个人，他的温柔只专属于真纪。不过有的时候，太为真纪着想了，以至于会忽略真纪本人的感受。就算是为了保护自己，也不能让自己喜欢的人受伤啊。

想想他在三大势力的压迫下，保护真纪，没有过人的头脑和坚定的爱也是很难做到的呢，为他点个赞！

推这条线的时候，艾西卡一直反复的说，“真纪，喜欢你，我爱你！”让我有一点疲劳呢！

有一个结局是作为黑莲会把艾西卡营救出来的报酬，真纪献出了自己的生命。五年后，成为了著名音乐家的艾西卡，他那段痛苦的内心独白，看的我都哭了。唯一还有个哭的地方，是卡尔洛线里真纪离开卡尔洛时内心的不舍独白。

这里的BE好像又有真纪沦为男娼的线呢。。。。

谜团（YY）

推玩艾西卡线，我觉得还有大量剧情留白啊，比如谢尔盖和尼古拉什么关系，应该是曾经相爱着的吧，可是到最后，对方都已经不是当初的那个人了，却还是死命的把彼此绑在一起，不愿松手。

还有爱德和亨利和伊贺啊，这三人啥关系啊，伊贺和亨利就是好朋友？爱德喜欢真纪还是喜欢他父亲亨利？（我的腐女心发作）

至于鸿上，比较清楚了，他应该是喜欢真纪的母亲羽月的，但是羽月不喜欢他啊，他对亨利有一些恨和抱怨的吧，但是亨利很器重他，还选他做继承人。

结束语

这游戏的语句描述？台词还是相当精彩的，就跟看小说似的，非常戳心。帮派之争剧情也很紧张，卡尔洛线里我真是头大，动不动就又被背叛了。

没错，我就是偏心，从人物描述篇幅就可以看出来来，我对卡尔洛是真爱，不说了我要再推一遍。
